About FermentIQ

AI-Powered Fermentation Monitoring Dashboard

What is FermentIQ?

FermentIQ is a cutting-edge, AI-powered real-time fermentation monitoring dashboard designed to provide precise control and insights into your fermentation batches. Using advanced machine learning algorithms and WebSocket-based streaming, FermentIQ delivers live updates on critical parameters like pH, temperature, and CO2 production.

Key Features

  • Real-Time Monitoring: WebSocket-based live streaming of batch data
  • AI-Powered Analysis: Machine learning algorithms compare your data against golden standards
  • Visual Analytics: Interactive charts for pH, Temperature, and CO2 metrics
  • Smart Alerts: Color-coded status indicators (Perfect, Acceptable, Concerning, Failed)
  • Historical Tracking: Event logs and AI-generated suggestions for batch optimization
  • Data Export: Download batch data as CSV for further analysis

Technology Stack

Frontend HTML5, CSS3, JavaScript, Chart.js
Backend FastAPI, Python, WebSockets
AI/ML scikit-learn, DTAIDistance, NumPy, Pandas
Communication MQTT, WebSockets, REST API

How It Works

FermentIQ continuously monitors your fermentation batches and compares them against a golden standard profile. The system uses Dynamic Time Warping (DTW) distance calculations to measure similarity, providing a quality score that indicates how closely your batch matches ideal fermentation conditions.

The AI system categorizes batches into four status categories:

Perfect: 95%+ match with golden standard
Acceptable: 90% to under 95% match
Concerning: 80% to under 90% match
Failed: Below 80% match
